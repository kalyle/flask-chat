# fun-chat

基于 flask 和 vue 的实时聊天项目

## backend

### 数据库迁移

#### windows

set FLASK_APP

> init
> `python -m flask db init`

> migrate
> `python -m flask db migrate`

#### linux

export FLASK_APP

> init
> `flask db init`

> migrate
> `flask db migrate`
> 技术栈：
> 功能：
> 好友上线提醒 OK
> 好友在线状态 OK
> 消息已读 OK
> 好友分组
